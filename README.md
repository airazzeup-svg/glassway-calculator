# Glassway — Калькулятор подвесных потолков

## Деплой на Vercel (без установки чего-либо на компьютер)

1. Зайди на [vercel.com](https://vercel.com) и зарегистрируйся (можно через GitHub, Google или email).
2. На главной странице нажми **Add New → Project**.
3. Выбери **Deploy without Git** / **Upload** (или раздел "Import" → перетащи папку проекта).
   - Если Vercel предложит подключить GitHub — это не обязательно, можно загрузить файлы напрямую через "Browse" или drag-and-drop всей папки `glassway-app`.
4. Vercel сам определит, что это Vite-проект, и предложит настройки:
   - **Build command**: `npm run build` (уже стоит по умолчанию)
   - **Output directory**: `dist` (уже стоит по умолчанию)
5. Нажми **Deploy** и подожди 1–2 минуты.
6. Получишь ссылку вида `https://glassway-calculator-xxxx.vercel.app` — это и есть твой калькулятор, открывается у любого человека без аккаунта.

## Как обновлять в будущем

Если попросишь Claude доработать калькулятор, он пришлёт новый файл `CeilingCalculator.jsx`. Чтобы обновить сайт на Vercel:
1. В панели Vercel открой свой проект.
2. Вкладка **Deployments** → кнопка **Redeploy**, либо просто загрузи обновлённую папку проекта заново (Vercel создаст новую версию, ссылка останется той же).

## Подключение своего домена (например calculator.glassway.group)

В настройках проекта на Vercel: **Settings → Domains → Add** — впиши нужный поддомен и следуй инструкции по настройке DNS-записи у регистратора домена.

## Локальный запуск (если понадобится проверить перед загрузкой)

```bash
npm install
npm run dev
```
Откроется на `http://localhost:5173`
